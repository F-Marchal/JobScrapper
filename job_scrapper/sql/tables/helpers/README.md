# Request Helper
This folder contains a number of class meant to be used to ease user's
database request. Those classes are heavily based on ../../filters/ behaviors.

## SqlRequestWrapper (`sql_request_wrapper`)
Parent of all request helpers, define base functionalities of request helpers.


---

## JobRequest (`job_request`)
Help with the request of Job object a joins with job related table such as
`Distances` or `JobExtraBase`
